# WIRE CONVERSIONS

The SVG's and many of the factory-spec wiring diagrams list the wire size in EU standard metric (mm2). The table below will help convert from that size to US AWG size.

| METRIC SIZE (MM2) | AWG SIZE |
| :---------------- | :------- |
| 0.05 | 30 |
| 0.08 | 28 |
| 0.14 | 26 |
| 0.25 | 24 |
| 0.34 | 22 |
| 0.38 | 21 |
| 0.50 | 20 |
| 0.75 | 18 |
| 1.50 | 16 |
| 2.50 | 14 |
| 4.00 | 12 |
| 6.00 | 10 |
| 10.0 | 8  |
| 16.0 | 6  |
| 25.0 | 4  |
| 35.0 | 2  |
| 50.0 | 1  |
| 55.0 | 1/0|
